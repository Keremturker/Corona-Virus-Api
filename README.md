# Corona Virus Api
Corona Virüs Api uygulaması ile ülkeler bazımda ölüm ve vaka sayılarını görebilirsiniz.

Veriler https://api.collectapi.com adresinden alınmaktadır.

# Features:
1. Kotlin
2. MVVM
3. Hilt
4. Coroutines
5. Retrofit
6. Jetpack Navigation (one activity, two fragments)
7. Shimmer
8. Kotlin Extension
